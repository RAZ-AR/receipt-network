## Правило источника истины

При противоречии документов приоритет такой:

1. последнее принятое решение в `DECISIONS.md`;
2. `PRODUCT_PRD.md`;
3. `MVP.md` для первого запуска;
4. специализированный документ;
5. старые заметки и backlog.
