# Screen Catalog

## Consumer application

### S01 — Welcome

Purpose: explain the value in one sentence and start the first scan.

Primary CTA: Scan receipt.

### S02 — Sign in and registration

Purpose: create or recover an account with minimum friction.

### S03 — Home

Purpose: show current value and the next best action.

Required modules:

- active reward goal;
- points balance;
- lottery progress;
- opportunity of the day;
- scan CTA.

### S04 — Scanner

Purpose: capture one or more readable receipts.

### S05 — Batch review

Purpose: review captured images before upload.

### S06 — Processing queue

Purpose: show a separate status for every receipt.

### S07 — Receipt result

Purpose: explain acceptance, points, tickets and goal progress.

### S08 — Receipt details

Purpose: show extracted data, status, history and dispute action.

### S09 — Wallet

Purpose: show points balance and immutable transaction history.

### S10 — Rewards catalog

Purpose: help the user select a meaningful attainable reward.

### S11 — Reward details

Purpose: explain price, stock, restrictions and redemption.

### S12 — Reward claim

Purpose: present QR, code, expiry and support path.

### S13 — Lottery overview

Purpose: show tickets, draw timing, prizes and rules.

### S14 — Lottery result

Purpose: show winners and claim state transparently.

### S15 — Opportunity details

Purpose: explain today's limited benefit and eligibility.

### S16 — Review form

Purpose: collect structured verified feedback.

### S17 — Receipt history

Purpose: find and filter previous receipts.

### S18 — Notifications

Purpose: show product events and value reminders.

### S19 — Profile and settings

Purpose: manage identity, preferences, privacy and language.

### S20 — Support

Purpose: resolve rejected receipts, reward and prize disputes.
